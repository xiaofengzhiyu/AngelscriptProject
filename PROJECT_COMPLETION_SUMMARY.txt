                          IMPORTANT NOTES
================================================================================

1. Historical references to `ANGELSCRIPT_TEST` and `ANGELSCRIPT_ISOLATED_TEST`
   should be treated as obsolete unless a compatibility layer is explicitly
   reintroduced.

2. Existing helper semantics matter:
   - `GetOrCreateSharedCloneEngine()` currently creates a shared Full engine
   - `AcquireCleanSharedCloneEngine()` resets that shared engine
   - `AcquireFreshSharedCloneEngine()` destroys shared/global state before reacquiring

3. Migration should therefore be driven by real behavior, not helper naming.
